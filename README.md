# Library Borrowing & Reservation System

A modern, professional frontend application for library management built with Next.js, React, and Tailwind CSS.

## Features

- **Modern UI Design**: Scandinavian/Swiss minimalism with premium aesthetics
- **Light/Dark Mode**: Full theme support with smooth transitions
- **Multi-language Support**: English and Turkish language support
- **Responsive Design**: Fully responsive across all device sizes
- **Dashboard Layout**: Professional sidebar and navigation structure

## Tech Stack

- **Next.js 14** (App Router)
- **React 18**
- **TypeScript**
- **Tailwind CSS**
- **next-intl** (Internationalization)

## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

### Build

```bash
npm run build
```

### Start Production Server

```bash
npm start
```

## Project Structure

```
├── app/
│   └── [locale]/
│       ├── login/          # Login page
│       ├── search/          # Search books page
│       ├── my-books/       # My books page
│       └── librarian/      # Librarian dashboard
├── components/
│   ├── ui/                 # Core UI components
│   ├── layout/             # Layout components
│   └── features/           # Feature components
├── hooks/                  # Custom hooks
├── messages/               # Translation files
└── public/                 # Static assets
```

## Pages

1. **Login Page** - Authentication with theme and language switchers
2. **Search Books** - Browse and search books with filters
3. **My Books** - View borrowed and reserved books
4. **Librarian Dashboard** - Manage borrowing and reservation requests
